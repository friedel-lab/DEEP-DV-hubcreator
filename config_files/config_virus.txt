config file for viruses to get extracted from GEO and SRA

This file consists of two parts: 
    1. List containing virus categories
    2. List containing abbreviation and synonyms for each virus of each category

1. Virus categories list:

Please enter your virus categories in the dotted box according to the following scheme:

> virus_category

> next_virus_category

------------------------------------------------------------------------------

> herpesvirus

> polyomavirus

> adenovirus

------------------------------------------------------------------------------


2. Virus synonym list:

Please enter your virus in the dotted box according to the following scheme:

> virus_abbreviation
virus_synonym_1, virus_synonym_2, ..., virus_synonym_x

> next_virus_abbreviation

Please pay attention to add new viruses into the correct virus_category. Each category is indicated by <virus_category> at the beginning of the section.
If you add virus synonyms of viruses from a new category please first add the indicator <virus_category> at the beginning.
